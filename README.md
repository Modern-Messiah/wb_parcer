# WB Parser

Простой парсер каталога Wildberries.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

По умолчанию запрос: `пальто из натуральной шерсти`.
Фразу задаётся в файле `settings.py` в константе `SEARCH_QUERY`.
